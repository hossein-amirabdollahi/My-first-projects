1_square-root:

	This program reads several positive numbers from the input and prints their square 
	root up to 4 decimal places.

	The first line of input indicates a number, n, which represents the count of numbers 
	for which it should calculate their square root. Each of the following n lines contains 
	a positive number. Please refer to the sample input and output for more information.

	Please note that the output is not rounded in any way. It will print exactly 4 decimal places.
	
Example:
Input:
4
1
2
19
100

Output:
1.0000
1.4142
4.3588
10.0000
