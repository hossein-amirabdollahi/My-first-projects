1_Hacking-through-Rainbow:

	This program helps you hack the password of each person, assuming the password is only 
	a four-digit number and you have the hash code of that password.

	For example, if a hash is generated from numbers 1000 to 9999, the program will be able 
	to output the password by entering the hash.

	Please note that the passwords range from 1000 to 9999.

Example:
Input:
danial,99b057c8e3461b97f8d6c461338cf664bc84706b9cc2812daaebf210ea1b9974
elham,85432a9890aa5071733459b423ab2aff9f085f56ddfdb26c8fae0c2a04dce84c

Output:
danial,5104
elham,9770
